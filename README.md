# JobIntel
JobIntel is an AI-Powered Job Insight Chrome Extension

##Description:
- The project is a Chrome extension that helps users analyze job postings using web scraping and artificial intelligence.
- When a user visits a job posting page, the extension extracts job details such as title, skills, and experience requirements using web scraping.
- The AI model then compares this information with the userâ€™s profile to generate insights, including skill match percentage, missing skills, and chances of job selection.
- Finally, the processed insights are displayed in the extension popup in a clear and structured format.
